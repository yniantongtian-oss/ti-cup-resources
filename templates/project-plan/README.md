# 电子设计竞赛项目管理模板

这套模板用于把“想法”推进为可复现、可答辩的工程项目。所有示例字段均为空白或虚构，不包含学校及个人信息。

## 文件

- `bom.csv`：器件、替代料、价格、库存和采购状态
- `test-record.csv`：测试条件、仪器、测量值、结论和证据路径
- `risk-register.csv`：技术、安全、进度和供应风险

## 使用顺序

1. 复制整个目录到自己的项目。
2. 先填写风险表，再确定采购和实验计划。
3. 每次实测都新增测试记录，不覆盖旧数据。
4. 图片、示波器截图和原始数据放在单独目录，并在 `evidence_path` 中填写相对路径。
5. 比赛结束后移除手机号、学号、内部群链接和未授权题目资料再公开。

CSV 使用 UTF-8 编码，Excel、LibreOffice 和多数脚本工具均可直接打开。
